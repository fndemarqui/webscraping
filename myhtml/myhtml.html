Conceitos básicos sobre HTML

HTML (HyperText Markup Language - Linguagem de marcação de texo em português) é uma linguagem utilizada para apresentarmos conteúdo na internet.

Um arquivo HTML nada mais é do que um arquivo de texto simples, que pode ser aberto e editado em qualquer editor de textos (vou sugerir aqui que vocês utilizem o RStudio, onde esse arquivo.html foi criado).

CSS é a linguagem utilizada para definir o estilo de um documento HTML. Mais especificamente, a linguagem CSS descreve como os elementos HTML devem ser exibidos.

Tags, elementos e atributos

Textos simples são transformados em um documento HTML através de tags (etiquetas) que podem ser interpretadas por um navegador.

Exemplos de tags incluem:

  • <h1>, <h2>,…,<h6>: maior heading, segundo maior heading, etc.
  • <p>: parágrafo
  • <ul>: lista desordenada
  • <ol>: lista ordenada
  • <li>: item individual de uma lista
  • <div>: divisão ou seção
  • <table>: tabela
